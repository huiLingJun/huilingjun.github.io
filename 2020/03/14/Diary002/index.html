huilingjun · 2020-3-14 · 次阅读

今日记录一下网上学习到的关于elemetUI的dialog弹窗拖拽的方法

The One VUE骚操作之this.$options.data()【大概一分钟】

    重置vue组件的data数据
在vue单文件组件里有时需要重置data中的数据，比如表单填写一半，用户想重新填写。

Way:

//重置表单方法
this.form = this.$options.data().form;
也可以通过给组件 $data 对象赋值来重置来重置整个 $data。
this.$data = this.$options.data();

结语：
vue组件可以通过 this.$options 对象获取你编写的任何方法，比如 created( ) 生命周期函数，发现vue新玩法，偷笑.jpg

The Three vue自动刷新的几种方法

Way:

1、window.location.reload(),是原生JS提供的方法，this.$router.go(0):是vue路由里面的一种方法，这两种方法都可以达到页面刷新的目的，简单粗暴，但是用户体验不好，相当于按F5刷新页面，会有短暂的白屏，相当于页面的重新载入。
2、通过路由跳转的方法刷新，具体思路是点击按钮跳转一个空白页，然后再马上跳回来：
        当前页面：
            <el-button type="primary" class="btn" @click="btnaction">摁我刷新页面</el-button>
            btnaction() {
            //    location.reload()
            //  this.$router.go(0)
                this.$router.replace({
                    path:'/empty',
                    name:'empty'
                })
            }
        空白页面：
            created(){
                this.$router.replace({
                    path:'/',
                    name:'father'
                })
            }
    当点击按钮时地址栏会有快速的地址切换过程。
3、控制<router-view></router-view>的显示与否，在全局组件注册一个方法，该方法控制router-view的显示与否，在子组件调用即可：
    App.vue:
        <template>
            <div id="app">
                <router-view v-if="isRouterAlive"></router-view>
            </div>
        </template>

        <script>
        export default {
            name: 'App',
            provide() {   // 注册一个方法
                return {
                    reload: this.reload
                }
            },
            data() {
                return {
                  isRouterAlive: true
                }
            },
            methods: {
                reload() {
                    this.isRouterAlive = false
                    this.$nextTick(function() {
                        this.isRouterAlive = true
                        console.log('reload')
                    })
                }
            }
        }
        </script>
    当前组件：
        inject: ['reload'], // 引入方法
        this.reload()   //调用方法

The Four 在表格里实现父菜单和子菜单的上下移 [2020/3/37 15:50]

Way

首先，在表格插入@row-click这个方法：
  <el-table
    v-loading="listLoading"
    :data="tableData"
    element-loading-text="Loading"
    border
    fit
    highlight-current-row
    row-key="id"
    :tree-props="{ children: 'children', hasChildren: 'hasChildren' }"
    @row-click="getRowData"
  >
然后拿到当前行的内容信息：
  getRowData(row) {
    this.rowInfo = row;
  },
上移方法：
  moveUp() {
    for (let i = 0; i < this.tableData.length; i++) {
      if (this.rowInfo.id == this.tableData[i].id) {
        if (i == 0) {
          // alert("已经是第一条，不可上移");
          return;
        } else {
          const selectData = this.tableData.splice(i, 1);
          this.tableData.splice(i - 1, 0, selectData[0]);
        }
      }
    }
    for (let i = 0; i < this.tableData.length; i++) {
      for (let j = 0; j < this.tableData[i].children.length; j++) {
        if (this.rowInfo.id == this.tableData[i].children[j].id) {
          if (j == 0) {
            // alert("已经是第一条，不可上移");
            return;
          } else {
            const selectData = this.tableData[i].children.splice(j, 1);
            this.tableData[i].children.splice(j - 1, 0, selectData[0]);
          }
        }
      }
    }
  },
下移方法：
  moveDown() {
    // 如果id 相等 那么 就拿到对应的下标
    for (let i = 0; i < this.tableData.length; i++) {
      if (this.rowInfo.id == this.tableData[i].id) {
        if (i == this.tableData.length - 1) {
          // alert("已经是最后一条，不可下移");
          return;
        } else {
          const selectData = this.tableData.splice(i, 1);
          this.tableData.splice(i + 1, 0, selectData[0]);
          return;
        }
      }
    }
    for (let i = 0; i < this.tableData.length; i++) {
      for (let j = 0; j < this.tableData[i].children.length; j++) {
        if (this.rowInfo.id == this.tableData[i].children[j].id) {
          if (j == this.tableData[i].children.length - 1) {
            // alert("已经是最后一条，不可下移");
            return;
          } else {
            const selectData = this.tableData[i].children.splice(j, 1);
            this.tableData[i].children.splice(j + 1, 0, selectData[0]);
            return;
          }
        }
      }
    }
  },
